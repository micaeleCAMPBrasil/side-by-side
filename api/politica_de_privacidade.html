Politicas de Privacidade 1. Introdução Bem-vindo ao Tuddo. Esta Política de Privacidade explica como coletamos, usamos, compartilhamos e protegemos suas informações pessoais ao utilizar nosso aplicativo. 2. Dados Coletados Coletamos os seguintes dados pessoais dos usuários: * Email: Utilizado para registro e login no app. Não é mostrado publicamente. * Nome: Utilizado para registro e login no app. Não é mostrado publicamente. * Username: Utilizado no app para identificação pública. Esses dados são coletados por meio de formulários. 3. Uso dos Dados Os dados pessoais coletados são utilizados para: * Registro no app e permitir que os usuários façam login. * Permitir que outras pessoas se conectem com o usuário através do username. 4. Compartilhamento de Dados Os dados pessoais dos usuários do Tuddo não são compartilhados com terceiros. 5. Segurança dos Dados Garantimos a segurança dos dados pessoais dos usuários através de medidas de segurança em nosso banco de dados e na proteção das informações dos usuários. 6. Direitos dos Usuários Os usuários podem acessar, alterar e excluir seus dados pessoais a qualquer momento através das configurações do app. 7. Cookies e Tecnologias de Rastreamento O Tuddo não utiliza cookies ou outras tecnologias de rastreamento. 8. Alterações na Política de Privacidade Os usuários serão informados sobre alterações na política de privacidade via notificação no app e email. 9. Contato Se você tiver dúvidas ou preocupações sobre esta política de privacidade, entre em contato conosco pelo email: tuddogramado@gmail.com
